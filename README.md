# PyPoll and PyBank

## PyBank
I created a Python script to analyze financial records (budget_data.csv) from a company. 

## PyPoll
I created a Python script that analyzes election data (election_data.csv) to determine the winner of an election. 

## Usage

To run PyBank:

```sh
cd PyBank
python main.py
```

To run PyPoll:

```sh
cd PyPoll
python main.py
```
